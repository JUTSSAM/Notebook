RBFN - 径向基函数网络

Dec 27, 2018   
概念解释：
vector <=> patern 即 向量和模式的概念在此文中是相同的。

RBF( 径向基函数网络 )

  • 将神经网络的设计看作是一个高维空间的曲线拟合（逼近）问题，采用不用的方法进行设计。
  • 学习 => 在多维空间中寻找一个能够最佳拟合(统计意义上的)训练数据的曲面。
  • 泛化 => 利用多维曲面对测试数据进行插值。（?多维空间中传统的严格插值法）

神经网络中的隐藏单元提供一个“函数”集，该函数集在输入模式（向量）拓展至隐藏空间时为其构造了一个任意的“基”；这个函数集中的函数就称为径向基函数。

最基本形式的径向基函数网络(RBFN)由三层构成：

  • 输入层：源点（感知单元），网络与外界环境连接。
  • 隐层：非线性变换
  • 输出层：现行，为输入层的激活信号提供响应。

模式可分性的Cover定理

将 [复杂的模式分类问题] 非线性地 投射 到 [高维空间] 将比投射到 [低维空间] 更可能是 [线性可分的]。

具有线性可分性的分类问题更容易解决，通过RBF网络作为模式分类器更易得到线性可分。

给定一族曲面，每个曲面都能自然地将输入空间自然地分成两个区域。用ae代表N个模式（向量）x1…xN的集合，每一个都分属于ae1和ae2中的一类。如果这族曲面中存在[一个]曲面能把分属于ae1和ae2的点分成两部分，则称这些点的二元划分关于这族曲面是可分的。

对于每一个模式向量，定义一个由一组实值函数fi组成的向量F。假设模式x是m0维向量上的点，则函数将m0维输入空间的点映射到新的m1维空间的对应的点上。将fi称为[隐藏函数]，它与前反馈神经网络中的隐藏单元起着相同的作用。生成的新的空间被称为[隐藏空间]或者[特征空间]。

如果在新的m1维空间中存在超平面使得fx可分，则该定义该超平面为隐藏空间（f空间）的[分离曲面]。

推论 - 一组随机指定的输入模式的集合在m1维空间中线性可分，它的元素数目的最大期望等于2m1。
